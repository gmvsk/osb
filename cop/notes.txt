✅ Step 1: WLST Script to Export ALL OSB Projects
Create this file on the OSB server (or copy it via Ansible):
📄 export_osb_projects.py
Pythonfrom java.lang import Systemfrom com.bea.wli.sb.management.configuration import SessionManagementMBeanfrom com.bea.wli.sb.management.configuration import ALSBConfigurationMBeanfrom com.bea.wli.sb.management.configuration import Exporterfrom com.bea.wli.sb.management.query import ProjectQueryfrom weblogic.management.scripting.utils import WLSTException# ---------------------------# Hardcoded configuration# ---------------------------ADMIN_URL = 't3://osb-admin-host:7001'OSB_USER = 'osb_shared_user'OSB_PASSWORD = 'password123'EXPORT_FILE = '/u01/osb_backup/osb_all_projects.jar'try:    print('Connecting to OSB Admin Server...')    connect(OSB_USER, OSB_PASSWORD, ADMIN_URL)    domainRuntime()    sessionName = 'osbExportSession'    sessionMBean = findService(        SessionManagementMBean.NAME,        SessionManagementMBean.TYPE    )    sessionMBean.createSession(sessionName)    print('Session created:', sessionName)    alsbMBean = findService(        ALSBConfigurationMBean.NAME,        ALSBConfigurationMBean.TYPE    )    query = ProjectQuery()    projects = alsbMBean.getProjects(query, sessionName)    projectNames = []    for p in projects:        projectNames.append(p.getName())    print('Projects found:', projectNames)    exporter = Exporter(alsbMBean)    exporter.exportProjects(projectNames, EXPORT_FILE, sessionName)    print('Export successful:', EXPORT_FILE)    sessionMBean.discardSession(sessionName)except WLSTException as e:    print('WLST Error:', e)finally:    disconnect()    exit()Show more lines

✅ Step 2: Ansible Playbook to Execute WLST
This playbook:

Switches to the oracle user
Runs WLST
Produces the export JAR

📄 osb_export.yml
YAML---- name: Export all OSB projects using WLST  hosts: osb_server  gather_facts: no  become: yes  become_user: oracle  tasks:    - name: Ensure backup directory exists      file:        path: /u01/osb_backup        state: directory        mode: '0755'    - name: Copy WLST export script      copy:        src: export_osb_projects.py        dest: /u01/osb_backup/export_osb_projects.py        mode: '0755'    - name: Run WLST OSB export      shell: |        /u01/oracle/Middleware/oracle_common/common/bin/wlst.sh \        /u01/osb_backup/export_osb_projects.py      register: wlst_output    - name: Show WLST output      debug:        var: wlst_output.stdoutShow more lines
